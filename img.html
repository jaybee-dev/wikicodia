IMG

Descrption

L'élément HTML <img> permet de représenter une image dans un document. Cet élément est un élément remplacé.

Tableau de attributs
alt src
Cet attribut définit le texte alternatif utilisé lorsqu'il est impossible d'afficher l'image (par exemple si l'URL est incorecte ou si l'image n'est pas encore téléchargée). L'URL de l'image. Cet attribut est obligatoire pour l'élément <img>. Pour les navigateurs qui prennent en charge srcset, src est considéré comme une image candidate dont la densité de pixel vaut 1x si aucune autre image avec cette densité n'est définie via srcset ou si srcset contient des descripteurs 'w'.

Exemple 
<img src="https://developer.mozilla.org/static/img/favicon144.png"
     alt="Le logo de MDN avec une silhouette de tête de dinosaure.">
