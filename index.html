HTML

Descrption

L'élément HTML <html> représente la racine d'un document HTML ou XHTML. Tout autre élément du document doit être un descendant de cet élément.

Tableau de attributs
manifest version xmlns
Définit l'URI d'un manifeste de ressources signifiant que les ressources devraient être mises en cache localement. Définit la version du document HTML Document Type Definition qui s'applique pour le document courant. Cet attribut n'est pas nécessaire car il est redondant avec l'information de version se trouvant dans la déclaration de type du document (doctype). Définit l'espace de noms XML du document. La valeur par défaut est "http://www.w3.org/1999/xhtml". Cet attribut est obligatoire dans un document XML et optionnel dans un document de type text/html.

Exemple 
<!DOCTYPE html>
<html lang="fr">
    <head>...</head>
    <body>...</body>
</html>
